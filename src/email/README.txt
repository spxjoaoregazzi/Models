Tutorial para usar o código:

1-Criar uma app password do google
  1.1-Tutorial em: https://support.google.com/accounts/answer/185833?hl=en
2-Baixar todos os requirements
  2.1-Abrir o prompt de comando
  2.2-Acessar esse repositório (cd "{caminho_repositório}/Modules/src/email"
  2.3-Executar "pip install -r requirements.txt"
3-Copiar e colar o arquivo get_email.py para o seu repositório
4-Importar o arquivo get_email no seu código python
  4.1-Escrever no início do seu código "import get_email"
